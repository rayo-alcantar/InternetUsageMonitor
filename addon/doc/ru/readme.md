# InternetUsageMonitor

## Описание

**InternetUsageMonitor** — это дополнение для экранного читателя NVDA, позволяющее пользователям отслеживать использование интернет-соединения в объёме отправленных и полученных данных. Этот инструмент особенно полезен для тех, кому необходимо вести учёт потребления трафика с течением времени — для управления пропускной способностью или просто из личного интереса.

## Как работает

После активации **InternetUsageMonitor** начинает отслеживать использование данных интернет-соединения пользователя. Мониторинг запускается и останавливается с помощью специальной комбинации клавиш (`Alt + NVDA + W`), которую можно изменить в разделе «Жесты ввода» в категории InternetUsageMonitor.

При запущенном мониторинге нажатие той же комбинации озвучивает объём использованного трафика и время (не останавливая таймер); двойное нажатие быстро друг за другом произнесёт данные и остановит мониторинг.

Также, благодаря [Анхелю Рейесу](https://github.com/reyes2005), добавлена возможность отслеживания использования интернета с уведомлениями.

* Настраиваемый жест в разделе «Жесты ввода»: `Shift + NVDA + W` — открывает интерфейс для установки лимита в мегабайтах.
* При достижении лимита каждые 30 секунд будет воспроизводиться голосовое и текстовое оповещение о превышении.
* Чтобы остановить оповещения, необходимо прекратить мониторинг.
